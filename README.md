# 星辰星迹
## 「星辰有迹，生活有序」

"星辰的轨迹，生活的印记" ✨

我们将浩瀚星空的秩序美学，融入日常时间管理，助你在纷繁生活中，找到属于自己的节奏。

---

## 核心功能

### 🌌 沉浸专注・番茄工作法
极简环形计时界面，搭配鸿蒙原生触感反馈，标准 / 冲刺 / 学习三种模式，助你高效捕捉每一段专注时光。

### 📝 清晰规划・任务管理
轻量化任务清单，支持快速添加、一键标记完成，清晰可视化的进度展示，让待办事项不再被遗忘。

### 📊 数据洞察・生活印记
精准记录设备状态与专注时长，生成多维度数据报表，让你清晰看见自己的时间轨迹与生活节奏。
